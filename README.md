# Object-Oriented Programming in Java (OOP_JS)

This repository contains Java programs that demonstrate the fundamental concepts of Object-Oriented Programming (OOP). It serves as a learning resource for understanding how OOP principles are implemented in Java through practical examples and exercises.

## 📚 Topics Covered

### Basic Java Concepts
- Variables and Data Types
- Operators
- Control Statements
- Loops
- Arrays
- Methods

### Object-Oriented Programming Concepts

#### Classes and Objects
- Creating Classes
- Object Instantiation
- Constructors

#### Encapsulation
- Access Modifiers
- Getters and Setters
- Data Hiding

#### Inheritance
- Single Inheritance
- Multilevel Inheritance
- Hierarchical Inheritance

#### Polymorphism
- Method Overloading
- Method Overriding
- Dynamic Method Dispatch

#### Abstraction
- Abstract Classes
- Abstract Methods

#### Interfaces
- Implementing Interfaces
- Multiple Inheritance using Interfaces

### Exception Handling
- Try-Catch Blocks
- Throw and Throws
- Custom Exceptions


## 🎯 Objectives

- Understand the principles of Object-Oriented Programming.
- Learn Java syntax and programming fundamentals.
- Implement real-world problem-solving using OOP concepts.
- Develop clean, reusable, and maintainable code.
